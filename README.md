# Start [Taro]

[Taro]: https://taro-docs.jd.com/

[![Status](https://img.shields.io/badge/Taro-3.x-brightgreen)](https://github.com/nervjs/taro)

## 前提准备

```zsh
yarn global add @tarojs/cli
```

## 创建项目

```zsh
taro init my-app
```

输出信息：

